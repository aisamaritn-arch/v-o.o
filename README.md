# SAMARITAN OS

&gt; Personal AI Operating System — v0.1.0

SAMARITAN OS is a modular, extensible AI backend designed to become the central intelligence for every digital task. It orchestrates LLMs, memory, tools, and knowledge rather than merely generating text.

## Philosophy

- **Build foundations first. Add intelligence later.**
- **Simplicity over cleverness.**
- **Explicit over implicit.**
- **Every module has one responsibility.**
